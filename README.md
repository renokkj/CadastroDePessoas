# Sistema de Cadastro de Pessoas e Produtos

Sistema desenvolvido em Java com integração a banco de dados MySQL para gerenciamento de pessoas e produtos.

O projeto permite realizar operações completas de CRUD (Create, Read, Update e Delete), aplicando conceitos de Programação Orientada a Objetos e persistência de dados via JDBC.

---

## Objetivo do Projeto

Desenvolver uma aplicação backend capaz de:

- Cadastrar pessoas e produtos;
- Consultar registros no banco de dados;
- Atualizar informações;
- Remover registros;
- Garantir persistência e integridade dos dados.

O sistema foi desenvolvido como projeto acadêmico na graduação em Engenharia de Software.

---

## Funcionalidades

### Pessoas
- Cadastro de pessoas;
- Listagem de registros;
- Atualização de dados;
- Exclusão de registros.

### Produtos
- Cadastro de produtos;
- Consulta por ID;
- Atualização de informações;
- Remoção de produtos.

---

## Tecnologias Utilizadas

- Java;
- MySQL;
- JDBC;
- SQL;
- Git.

---

## Conceitos Aplicados

- Programação Orientada a Objetos (POO);
- Modelagem de banco de dados;
- Conexão com banco via JDBC;
- Uso de PreparedStatement para segurança;
- Estruturação em camadas (Model / DAO);
- Tratamento de exceções;
- Operações CRUD.

---

## Estrutura do Banco de Dados

O sistema utiliza MySQL para armazenamento dos dados, com tabelas separadas para:

- Pessoas;
- Produtos.

As tabelas foram criadas utilizando comandos SQL no MySQL Workbench.

---

## Como Executar o Projeto

1. Clonar o repositório;
2. Configurar o banco de dados MySQL;
3. Atualizar as credenciais de conexão no arquivo de configuração;
4. Executar a aplicação via IDE (IntelliJ, Eclipse ou similar).
